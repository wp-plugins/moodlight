== Description ==

Moodlight allows your visitors to add their mood on posts via comments.
You can customize your posts and comments depending on visitors moods.
The visitors moods are translated by colors, so it is easy to know if peoples are agree or against the post subject.

----

Moodlight Permet à vos visiteurs de laisser une appréciation sur un article via les commentaires.
Vous pouvez personnaliser vos articles et vos commentaires en fonction de l'humeur des visiteurs.
Les appréciations des visiteurs sont traduites par des couleurs, il devient alors très simple de savoir si les gens sont pour ou contre le sujet d'un article.

== Installation ==
1. Send MoodLight to your blog, and activate it in the plugin page of Wordpress admin.
2. You can configure it through the "Moodlight" tab in Options.
3. Add the template functions to your current template, you'll find a documentation at "Options > MoodLight > Documentation" tab.

--

1. Envoyez MoodLight sur votre blog, et activez le dans la page plugin de l'administration de WordPress.
2. Vous pouvez le configurer grâce a l'onglet "Moodlight" dans Paramètres.
3. Ajoutez les fonctions de templates dans votre template actuel, vous retrouverez une documentation dans l'onglet "Paramètres > Moodlight > Documentation"
